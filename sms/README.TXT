문서작성일
통큰아이 SMS팀(sms@goodintert.co.kr)
작성일자:2008년 11월 24일
 
* 본 압축된 PHP 프로그램은 통큰아이에서 제공하는 SMS 예제입니다.(PHP 버전입니다.) 

* 특징:
통큰아이 SMS 호스팅은 XML웹서비스를 이용합니다. 따라서 별도의 서비스 포트가 필요하거나 
방화벽의 제약을 받지 않습니다.

간단한 함수호출만으로 SMS를 발송 하실수 있습니다.


*이용방법
다음 받으신 sms_sample_php_utf8.zip의 압축을 푸신후
고객님의 서버에 업로드를 해주십시요.
서버 설정은 php설치시 xml 옵션이 들어가 있어야 합니다.(대부분의 호스팅 업체는 xml 사용가능)

sms_process.php 파일에 $sms_id,$sms_pwd 정보를 부여받으신 정보로 변경하시면 바로 테스트 가능합니다.


* 유의점
1. 통큰 아이 SMS 호스팅에서는 고객님께 발급해드린 SMS 아이디와 SMS 패스워드(회원 아이디와 별도임)
   로 서비스를 이용하시므로 본 아이디와 패스워드가 유출되지 않도록 각별히 주의 해 주십시요.

2. 과도한 스팸 SMS 발송은 법적 제재 조치의 대상이 됩니다. 스팸 방지 가이드라인을 꼭 참조하시어
   불필요한 광고성 SMS가 발송 되지 않도록 유념해 주시기 바랍니다.
